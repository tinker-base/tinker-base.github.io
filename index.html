Relational Database

Each backend has a full PostgreSQL database for your application's relational data.

RESTful API

Forget about writing backend CRUD routes. PostgREST turns your PostgreSQL database directly into a RESTful API.

Self-hosted in AWS

Tinker deploys into your AWS account, granting you full autonomy over your backends. Customize and fine-tune according to your needs.

UI for SQL

Tinker's dashboard table editor makes setting up PostgreSQL easy and intuitive.
